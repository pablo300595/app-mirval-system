# Development Setup
Prerequisites

* Install [Node.js](https://nodejs.org/en/download) version 16 or higher.
* Install [Angular CLI](https://angular.io/cli) version 15.0.5.
* Read/Write Access to this repo.

## Quickstart
Installing node_modules dependencies:

`sudo npm install`

Run startup command:

`npm run start`
